Softwares

A HF Engenharia utiliza os seguintes softwares de projeto:

- TQS: software para projeto, dimensionamento e detalhamento de estruturas de concreto armado, protendido e alvenaria estrutural, possui compatibilidade com ferramentas tipo BIM, geração de arquivos no formato 3D

- EBERICK: software para projeto, dimensionamento e detalhamento de estruturas de concreto armado.

- CYPE 3D: software para projeto, dimensionamento e detalhamento de estruturas metálicas, engloba normas nacionais e internacionais.
