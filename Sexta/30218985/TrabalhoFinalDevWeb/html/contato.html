Olá sou Kayky Trevizan, estudante do 3° periodo Análise e Desenvolvimento de Sistemas na UP, e esse é um projeto da matéria de Desenvolvimento Web Básico, onde descrevo de uma forma resumida um pouco sobre as principais linguagens de programação utilizadas no mercado. Nesse projeto mostro quais são os principais recursos de cada linguagem e suas aplicações. Em cada página também mostra um simples exemplo de um programa hello world em cada linguagem. Para desenvolver esse projeto foi utilizado basicamente tudo o que foi visto ao longo do semestre. Utilizei apenas HTML, CSS e JavaScript puros para desenvolver esse projeto.
