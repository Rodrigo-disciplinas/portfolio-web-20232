Algumas tags interessantes

  • <strong>

    • Exemplo didatico da utilização correta da tag strong. É utilizada para destacar e enfatizar um texto importante em HTML, melhorando a acessibilidade e a compreensão do conteúdo.

  • <blockquote>

    • “A educação é a arma mais poderosa que você pode usar para mudar o mundo.”

    Nelson Mandela

  • <abbr>

    • O WWW revolucionou a forma como compartilhamos informações.

  • <em>

    • Amor é um sentimento poderoso que conecta as pessoas.

  • <sup>

    • A água ferve a 100°C ao nível do mar.

  • <sub>

    • A fórmula química do dióxido de carbono é CO2.

  • <code>

    • Para imprimir uma mensagem no console, use o comando console.log().

  • <mark>

    • Destaque a palavra importante neste parágrafo.

  • <pre>
    •                         function olaMundo() {
                            console.log("Olá, mundo!");
                        }
  • <time>

    • A data de lançamento do novo produto é 15 de junho de 2023.
