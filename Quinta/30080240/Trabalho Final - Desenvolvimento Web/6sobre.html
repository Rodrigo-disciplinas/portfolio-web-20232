Sobre nós

Tibia RPG é o maior fansite brasileiro sobre o game Tibia, que é um dos MMORPGs mais antigos em plena atividade.

Este é um site que busca angariar cada vez mais jogadores fiéis ao Tibia.

Esse projeto se iniciou como um Fan Site, para um projeto desenvolvido na faculdade, mais especificamente na matéria de Desenvolvimento Web Básico, ministrada pelo Prof° Rodrigo Machniewicz Sokulski, que incentivou e ofertou as ferramentos necessárias para que essa ideia saísse do papel.
